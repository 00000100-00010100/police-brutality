# Police Brutality

This subreddit is a repository for evidence of police brutality during the George Floyd protests of 2020.

This was started on [r/2020PoliceBrutality](https://www.reddit.com/r/2020PoliceBrutality) and moved to Github to enable more people to participate in the collection of resources.
